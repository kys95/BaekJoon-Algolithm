# 문제

1. 피는 시기& 지는 시기가 결정된 꽃들중에서 3월1일 부터 11월30일까지 매일 한 가지 이상 피어 있도록 해야한다.
2. 가능한 한 가장 적은 꽃들의 수를 출력하는데 조건을 만족할 수 없다면 0을 출력한다.



# 해결과정

1. 꽃이 피고 지는 날짜의 월에 100을 곱하고 일을 더하면 mmdd와 같은 수가 된다.

   ex) 1 1 5 31 -> 101, 531 -> 1월 1일에 피어서 5월 31일에 진다.

2. 꽃이 피는날짜가 3월1일 보다 이르거나 같을 때, 지는 날짜가 가장 긴 꽃을 선택해야 최소한의 꽃을 심을 수 있다.

3. 꽃이 가장 늦게 지는 날짜를 다시 기준(3월1일처럼) 으로 갱신하여 다음 날짜들을 비교한다.
